# Simpel-Keystroke-Keyboard
Keystrokes on python, when u press "w,a,s,d" it will show the key that you press on keyboard

---
> HOW TO USE : </br>
```
1. pip install keyboard and pip install tkinter
2. run the file `py main.py`
```
